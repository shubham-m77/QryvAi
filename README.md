# QyrvAI

QyrvAI is an AI-powered career assistant built to help users prepare smarter for job opportunities with personalized career tools, resume support, interview preparation, and industry insights.

## Overview

QyrvAI helps candidates improve their job readiness by providing AI-driven guidance based on their profile, skills, experience, and career goals. The platform is designed with a clean user experience, modern UI, and practical tools for career growth.

## Features

- AI-powered career guidance
- Resume and profile enhancement
- Mock interview preparation
- AI-generated cover letters
- Industry insights and skill recommendations
- Google authentication
- Personalized dashboard
- Modern and responsive interface

## Tech Stack

**Frontend:** Next.js, TypeScript, Tailwind CSS  
**Backend:** Next.js API Routes, Server Actions  
**Database:** PostgreSQL with Prisma  
**Authentication:** NextAuth.js  
**AI:** Gemini API  
**Deployment:** Vercel
